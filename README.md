# Vrund Patel Portfolio

A production-ready personal portfolio built with React, Vite, Tailwind CSS, Framer Motion, React Icons, and smooth scrolling.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Personalization

- Replace `src/assets/vrund.jpg` with the final profile photo.
- Replace `public/resume.pdf` with the final resume.
- Replace `vrund@example.com` in `src/components/Contact.jsx` with the final email address.

## Vercel

The project includes `vercel.json` with an SPA rewrite. Push to GitHub, import the repository on Vercel, and use the default Vite build command:

```bash
npm run build
```
